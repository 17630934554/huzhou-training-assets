Chapter 01

系统简介

把设备组的 SOP 培训资料做成"扫码即看"的网页系统。

🎯 解决什么问题

  • 找不到：纸质 SOP 散落各处，新人不知道放哪
  • 看不懂：纯文字描述不直观，现场作业要反复打电话问
  • 不更新：设备换型 / 流程调整后，老资料没人维护
  • 不统一：飞书 / 微信 / 钉钉 各种版本，链接 30 天就失效

📦 系统组成

  • 5 份 SOP 培训页（lesson_1 ~ lesson_5）视频 + 重点分析 + 截图
  • 5 个 B 站教学视频（湖州仓现场实拍）
  • 5 张 QR 卡（贴设备旁 / 仓库现场）
  • 1 行命令新增培训资料（add_training.py）
  • 永久 hosting（Cloudflare Pages 全球 CDN）
核心承诺
扫码即看 / 永久有效 / 全员可访问（本人 / 内部 / 外部 / 匿名） / 自助更新 5 分钟上线

Chapter 03 · A

自助维护 · 拍视频 + 生成培训页

① 现场拍视频（5 分钟）

📱 拍视频要点

  • 手机竖屏拍摄（现场讲解更自然）
  • 从问题现象开始拍，故障 / 异常先入镜
  • 边操作边讲解，步骤清晰不跳跃
  • 总时长控制在 3-6 分钟
  • 导出文件名改有意义（不要 mmexport*.mp4）

🎬 上传 B 站

  • B 站账号：喃呵咦檬（UID 405067223）
  • 视频分类：科技 · 工业 · 自动化
  • 标题模板：湖州仓·[设备名]：[故障名] 处理
  • 上传完成后记录 BV 号

② 写 lesson HTML（10 分钟）

用现有 lesson_N.html 当模板复制修改：顶部嵌入 B 站 iframe（用 BV 号生成嵌入 URL），下方写"重点分析 + 步骤拆解 + 截图"。完成 1 份标准 lesson HTML（建议大小 800KB-1.5MB）。

③ 一行命令生成（10 秒）

# 1 行命令完成 3 件事：复制 HTML + 重生 index.html + 出 QR 卡 python3 add_training.py \ --id 6 \ --html my_lesson_6.html \ --title "自动分拣机：托盘更换 SOP" \ --bvid BV1xxxxxxxx \ --duration "4:30" # 自动产出： → hosting/lesson_6.html → hosting/index.html (导航页自动加新卡片) → hosting/qr/hosting_6_QR卡.png

Chapter 03 · B

自助维护 · 部署上线

④ 推送到 GitHub（30 秒）

python3 scripts/push_github.py \ --token <你的PAT> \ --owner 17630934554 \ --repo huzhou-training-assets # 或在 GitHub 网页端： # 1. 打开 https://github.com/17630934554/huzhou-training-assets # 2. 上传文件 / 编辑 README → 提交

⑤ Cloudflare Pages 自动部署（1-2 分钟）

GitHub push main → Cloudflare Pages webhook 触发 → 自动构建 → 自动部署。无需人工干预。

⑥ 验证部署（30 秒）

python3 scripts/verify_urls.py # 期望输出：所有 5+ 份 lesson + index.html + QR 卡 = HTTP 200
完整时间表

注意
QR 卡和 lesson HTML 永久禁用 "AI 生成" / "湖州仓·设备组" 这两个词（add_training.py 自动校验）。

Chapter 04

BTN 品牌汇报生成器

除了培训资料，还有 A2 BTN 品牌汇报系统 — 给集团 / 物流部做正式汇报用。

📊 完整 H1 汇报

  • btn-report-2026h1.html · 8 页
  • 封面 / 目录 / 部门介绍 / 上半年成绩 / V3 系统 / 改造案例 / 下半年规划 / 结束
  • 贝泰妮深蓝 + 金色克制点缀 + 14px 圆角
  • 102 个可编辑元素，文字可改后自动保存到 localStorage

🛠 生成器

  • btn_html_slides.py · 10 种页面模板
  • YAML 配置 → HTML 输出
  • 支持：封面 / 目录 / 章节 / KPI / 双栏 / 视频 / 计划
  • 单文件自包含（CSS / JS 全内联）
  • 文字可编辑 + 视频内嵌 + 键盘翻页

用法

# 1) 编辑 YAML 配置 vim btn_report_2026h1.yaml # 2) 生成 HTML python3 btn_html_slides.py \ --config btn_report_2026h1.yaml \ --output btn-report-2026h1.html # 3) 浏览器打开（键盘 ← → 翻页，T 切主题，F 全屏） open btn-report-2026h1.html
